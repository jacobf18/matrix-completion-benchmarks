# Demo Data

Optional demo files for the website's "Load Demo" buttons.

Supported CSV names:
- `synthetic_denoise_results.csv`
- `noise_sweep_results.csv`
- `hankel_results.csv`

Supported JSON names:
- `tabular_soft_impute_eval.json`
- `tabular_mi_eval.json`
- `tabular_eval.json`

The website will attempt these filenames in order and load whichever files are found.
